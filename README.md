# Sistema de asistencias

Sistema basico de asistencias de visitas para personal.

Despliegue:

```bash
npm install-all
npm run dev
```

DB: Almacenada en Supabase, las credenciales estan en texto plano, con lo cual se sugiere securizarlas para un deploy.
